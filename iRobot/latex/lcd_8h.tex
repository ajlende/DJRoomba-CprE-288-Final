Prints a string to the lcd; Google \char`\"{}printf\char`\"{} for documentation.

Mimics the C library function printf for writing to the L\-C\-D screen. The function is buffered; i.\-e. if you call lprintf twice with the same string, it will only update the L\-C\-D the first time.

Google \char`\"{}printf\char`\"{} for documentation on the formatter string.
